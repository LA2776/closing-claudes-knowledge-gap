FlutterFlow is soaring with new features this spring! This product update brings the ability to transform JSON responses into data types and the introduces custom image map markers. Let's dive in and explore what's new!
Custom Image Map Markers 📍
Elevate the visual appeal of your maps in FlutterFlow with the new Custom Image Map Markers feature, where you can now use your own images as markers on Google Maps. Set unique, eye-catching icons to provide a personalized map experience that truly stands out!
Data Types from JSON 🗃️
Speed up your development process by generating data types directly from JSON responses in FlutterFlow. This user experience enhancement allows for instant and accurate data structure creation, propelling your development forward with unmatched precision.
