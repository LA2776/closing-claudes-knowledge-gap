New Features For You ✨
We're kicking off February with a suite of new features designed to boost your app's functionality and scalability. From parameters in callback actions to a now overpowered DataTable ⚡️, we've got a bunch of updates for you.
Make sure to read to the end for a sneak peak at a big release coming soon! 👀
Callback Actions with Parameters ℹ️
Callback actions can now take parameters. Say goodbye to the hassle of using App State for managing specific callbacks. This improvement opens the door to creating powerful component libraries that teams can easily share and use across different projects.
Get ready to streamline your workflows and build even more amazing projects with ease!
Upgraded DataTable 🛠️
The DataTable widget is now more powerful and versatile for building complex, data-driven applications. Features now include column sorting for better data organization, toggle select for easier data manipulation, and customizable pagination among other exciting improvements! 
New Color Picker 🎨
Exciting news for design enthusiasts! We've introduced a brand new color picker designed to streamline your workflow and ensure design consistency. Now featuring a handy search function, finding the ideal shade for borders or containers is faster than ever.
Export Project as Flutter Module 📦
We've introduced the ability to directly export projects as Flutter modules using the FlutterFlow CLI! This powerful new functionality is designed to seamlessly bridge the gap between FlutterFlow and your existing applications, allowing for the integration of rich, customizable, and interactive features developed within FlutterFlow.
Other Improvements 🏗️
You can now dynamically set the pin length for PinCode Field Widget.
We've added a new action to clear multi-select dropdowns.
You can now create your own custom IDs for Firebase documents using the "custom ID" toggle.
We've implemented the UMP SDK for Admobs.
Added an option to remove the Google Permissions required for Automated Firebase Setup.
Please visit the GitHub Issue Tracker for the most up to date status of all reported bugs and fixes.
