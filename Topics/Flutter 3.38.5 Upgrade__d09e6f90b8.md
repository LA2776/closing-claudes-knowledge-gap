We will be upgrading the underlying Flutter version in FlutterFlow to 3.38.5 on Tuesday, February 3.
This release includes some significant updates to Flutter itself, plus updates to dozens of third-party packages used across FlutterFlow, now providing more stability with all third-party integrations.
We expect that most projects will continue to work as expected, but you may need to make changes to custom code or dependencies.
For more information, check out our Migration Guide.
And if you run into issues with custom dependencies, check out our video on How to Resolve Dependency Issues.
