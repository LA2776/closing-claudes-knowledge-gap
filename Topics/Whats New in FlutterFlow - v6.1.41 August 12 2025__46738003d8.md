Advanced Content: Super Apps in FlutterFlow with Libraries
Super Apps handle multiple functionalities — from food delivery and hotel bookings to payments and shopping — all in one app. In FlutterFlow, you can use libraries to build super apps modularly. Learn more about Super Apps in this video and blog post. 
🏗️Improvements & Bug Fixes 
We’ve been focused on reliability and stability in FlutterFlow. Here are some of the improvements and bug fixes in this week’s release: 
Added a toggle for enabling safe area for Google navigation bar
Added fixes which were causing the desktop app to crash for large projects 
Updated BrainTree to most recent iOS version
Improved error detection for invalid data types in projects
Annotations Dialog: Fixed an issue where tapping outside the dialog would close it without saving changes; now, unsaved changes prevent dismissal.
Fixed a bug causing test mode compilation errors for projects using the Stripe action
Fixed Android build issue for projects using flutter_quill and flutter_keyboard_visibility 
Fixed a bug that prevented custom text styles from being renamed
