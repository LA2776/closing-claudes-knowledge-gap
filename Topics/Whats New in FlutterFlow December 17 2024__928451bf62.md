Environment Specific Package Name
Building on last week’s update for environment-specific web and mobile deployment settings, you can now configure the package name for your app per environment. These features together make it simple to deploy your FlutterFlow app across multiple testing environments—like publishing an unlisted app to the iOS App Store for beta user testing.
See the documentation for more details.
Control the Swipe Direction for Swipeable Stacks
You can now control the swipe direction for SwipeableStack widget in your app. Whether you want users to swipe up, down, left, or right, this feature gives you full control over how users interact with your swipeable content.
Support for Supabase OR filters
You can now use OR filters in your queries, making it easier to retrieve data that matches multiple conditions. See the documentation on Supabase for more details.
