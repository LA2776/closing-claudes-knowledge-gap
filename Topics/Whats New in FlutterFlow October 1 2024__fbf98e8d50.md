Visual Studio Code Extension
The FlutterFlow Visual Studio Code extension is designed to enhance the developer experience by integrating FlutterFlow’s visual app-building platform with the code editing capabilities of Visual Studio Code. 
Now, you can edit custom code in a full integrated development environment - which allows you to see the entire codebase, benefit from reliable realtime error detection and autocomplete, plus take advantage of ecosystem tooling like Flutter’s extension or Microsoft’s Copilot.
For details on installing and using the Visual Studio Code Extension, see the documentation. 
FlutterFlow AI Tooling
Take full advantage of some of the AI tooling we showed at FFDC! We’re excited to launch several new tools, available in the new FlutterFlow AI Tools menu in the Toolbar. 
Page Autocomplete
Page Autocomplete intelligently adds relevant widgets to a page based on the pages’s context, making it easy to complete partially built pages
Sketch to Component
Sketch to Component allows you to add a sketch of your component. This sketch can include outlines of widgets or layouts to incorporate, and it can also include annotations that tell the AI tool more about the design you have in mind
Enhanced Page & Component Generation
We’ve also made various improvements to our existing Page and Component generation. Now when you generate a Page or a Component from a prompt, you should see more visually appealing results.
