🎊 We have some exciting updates to kick off the new year!
Leverage GitHub Repos for Custom Code Dependencies
You can now use public and private GitHub repositories as package dependencies for custom code in FlutterFlow! This powerful feature lets you host your team's code and seamlessly reference it in your custom actions, functions, and widgets. 
See the documentation for more details.
Page-Gen v2
We’ve enhanced our AI Page Generator to deliver more consistent, higher-quality UI designs. While this new version takes a bit longer to generate results, the improved outcomes make it worth the wait. For now, it’s available as an experimental feature, and you can toggle it on or off as you prefer.
We’d love to hear your feedback on the new version as we continue refining it!
Libraries Available in FlutterFlow Marketplace
A few weeks ago, we introduced the ability to add Libraries in FlutterFlow Marketplace, and the response has been incredible! Developers have added so many creative and useful libraries, making it easier to extend your app's functionality.
Here are a few must-try Libraries:
Adapty: Optimize your in-app subscriptions.
BuildShip: Seamlessly connect to your backend.
Shadcn: Beautiful, ready-to-use UI components.
Dive in and explore the possibilities! We can’t wait to hear which ones you love the most. 🚀
